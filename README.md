## Overview

**GUIDED** is a full-stack mentorship platform that streamlines the journey from career exploration to structured mentorship. Candidates complete a guided onboarding flow, receive an **AI-generated career roadmap** (powered by Google Gemini), browse and select verified mentors, and then track their progress through a personalised dashboard with sessions, action items, and skill-gap analysis.

Mentors go through an admin-verified onboarding process, manage their mentees, accept/decline mentorship requests, and have sessions automatically scheduled via **Google Calendar** integration.

Admins have a dedicated dashboard for platform analytics, mentor verification, mentorship monitoring, and manual mentor-mentee allocation.

## Getting Started

### Prerequisites

| Tool | Version |
|------|---------|
| **Node.js** | 18+ |
| **npm** | 9+ |
| **Python** | 3.10+ |
| **MongoDB** | 6+ (running on `localhost:27017`) |

### 1. Backend

```bash
cd backend

# Create and activate virtual environment
python -m venv .venv
.venv\Scripts\activate        # Windows
# source .venv/bin/activate   # macOS/Linux

# Install dependencies
pip install -r requirements.txt

# Start the server
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

The API will be available at **http://localhost:8000**  
Interactive docs at **http://localhost:8000/docs**

> **Note:** On first startup, the backend auto-seeds demo data (users, mentors, candidates) into MongoDB if the `users` collection is empty.

### 2. Frontend (Candidate & Mentor Portal)

```bash
cd frontend

# Install dependencies
npm install

# Start dev server (Turbopack)
npm run dev
```

Available at **http://localhost:3000**

### 3. Admin Frontend

```bash
cd admin-frontend

# Install dependencies
npm install

# Start dev server (Turbopack)
npm run dev
```

Available at **http://localhost:3001** (auto-increments if 3000 is taken)

## API Reference

The backend exposes **36 REST endpoints** across 5 groups:

### Auth (`/auth`)
| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/auth/signup` | Register a new user |
| `POST` | `/auth/login` | Login â†’ returns JWT token |
| `GET` | `/auth/me` | Get current user profile |

### Candidate (`/candidate`)
| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/candidate/status` | Check onboarding/workflow stage |
| `POST` | `/candidate/onboarding` | Save career profile |
| `POST` | `/candidate/generate-roadmap` | AI roadmap generation |
| `POST` | `/candidate/regenerate-roadmap` | Force re-generate roadmap |
| `POST` | `/candidate/checkout` | Finalise mentor + sessions |
| `GET` | `/candidate/workflow/{id}` | Full dashboard data |
| `POST` | `/candidate/toggle-action` | Toggle action item completion |
| `POST` | `/candidate/complete-session` | Mark session as completed |

### Mentor (`/mentor`)
| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/mentors` | Search/filter mentor catalogue |
| `GET` | `/mentors/{id}` | Single mentor profile |
| `GET` | `/mentor/dashboard` | Mentor dashboard data |
| `POST` | `/mentor/accept` | Accept mentorship request |
| `POST` | `/mentor/decline` | Decline mentorship request |
| `POST` | `/mentor/verify-submit` | Submit LinkedIn for verification |
| `GET` | `/mentor/verification-status` | Check verification status |

### Admin (`/admin`)
