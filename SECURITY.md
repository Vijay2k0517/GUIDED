---

## 📢 Reporting a Vulnerability

If you discover a security vulnerability in Guided Application, please report it responsibly and privately.

### 📩 How to Report

Send an email to:

**Email:** your-email@example.com  
**Subject:** `[Security] Guided Application Vulnerability Report`

Please include the following details:

- Description of the vulnerability
- Steps to reproduce the issue
- Expected behavior
- Actual behavior
- Screenshots or logs (if applicable)
- Your contact details (optional)

---

## ⏳ Response Timeline

- Initial acknowledgment: **Within 48 hours**
- Status update: **Within 5 business days**
- Fix deployment: Based on severity level

Critical vulnerabilities will be prioritized.

---

## 🔒 Responsible Disclosure Guidelines

We kindly request that you:

- Do **not** publicly disclose the issue before it is resolved.
- Provide reasonable time for investigation and patching.
- Avoid exploiting the vulnerability beyond proof-of-concept testing.

We appreciate responsible disclosure and may acknowledge contributors (if desired).

Guided Application follows industry best practices including:

- Data protection and privacy measures
- Secure API handling
